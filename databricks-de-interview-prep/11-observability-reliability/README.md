## 5-Minute Revision Cheat Sheet

- **Metrics:** Row counts, duration, success/fail. Log to table or external (Datadog, etc.).
- **SLA:** Target completion time. Monitor; alert on breach.
- **Pattern:** Job writes metrics; monitoring queries metrics; alerting on threshold.
