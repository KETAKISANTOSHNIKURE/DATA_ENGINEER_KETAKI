# Databricks Data Engineer Interview Preparation

**Target:** 3+ years experience | **Focus:** No fluff, interview-only content | **Coverage:** Every topic asked in real interviews

---

## How to Use This Repo

1. **Start with** `cram-plans.md` — pick your timeline (1 week / 3 day / 1 day)
2. **Work through** topics in folder order (01 → 11)
3. **Drill** `99-question-bank/` before interview
4. **Final pass** `final-interview-checklist.md` 24 hours before
