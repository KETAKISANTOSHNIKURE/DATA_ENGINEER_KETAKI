## 5-Minute Revision Cheat Sheet

- **Repos:** Git in Databricks; sync notebooks, folders. Branch, PR.
- **CI/CD:** Build → test → deploy. Databricks Asset Bundles (DAB) or API.
- **Promotion:** Dev → Staging → Prod. Use catalogs, clusters, secrets per env.
